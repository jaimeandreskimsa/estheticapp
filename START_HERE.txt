💡 LOS 4 MÓDULOS PRINCIPALES

Consulta → Diagnóstico → Plan → Presupuesto → (Aceptación) → Sesiones

MÓDULO 1: CONSULTA
├─ Captura motivo (texto + voz)
├─ Anamnesis (checkboxes + texto)
├─ Evaluación física (mapa corporal)
└─ Fotos clínicas

MÓDULO 2: DIAGNÓSTICO (IA)
├─ Análisis de datos capturados
├─ Genera diagnóstico clínico
├─ Crea resumen paciente
└─ Marca zonas visuales

MÓDULO 3: PLAN (IA + Médico)
├─ Sugiere procedimientos
├─ Calcula cantidades
├─ Ordena sesiones
└─ Crea timeline

MÓDULO 4: PRESUPUESTO (Automático)
├─ Busca precios de productos
├─ Calcula totales
├─ Calcula margen
└─ Versión paciente + profesional
