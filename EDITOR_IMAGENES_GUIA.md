## 🎨 Herramientas Avanzadas de Anotación

### 🆕 Nuevas Herramientas Profesionales

#### ✏️ Dibujo Libre (Freehand)
- Dibujar a mano alzada sobre la imagen
- Ideal para contornos faciales
- Control total del trazo
- Simular incisiones o áreas de tratamiento

#### 📐 Medición Profesional
- Medir distancias con precisión
- Marcadores en los extremos
- Muestra distancia en píxeles
- Útil para simetría facial

#### 🔢 Marcadores Numerados
- Numerar puntos de tratamiento secuencialmente
- Auto-incremento del contador
- Personalizable (aumentar/disminuir número)
- Ideal para protocolos paso a paso

#### ⬜ Rectángulos
- Delimitar áreas de tratamiento
- Marcar zonas específicas
- Útil para antes/después

## 🌈 Ajustes de Imagen Profesionales

### ☀️ Brillo (50% - 150%)
- Iluminar áreas oscuras
- Resaltar detalles
- Compensar iluminación deficiente

### ⚫ Contraste (50% - 150%)
- Definir características faciales
- Resaltar estructuras óseas
- Mejorar definición de zonas

### 💧 Saturación (0% - 200%)
- Modo B&N (0%) para análisis estructural
- Reducir para enfocarse en forma
- Aumentar para destacar tonos de piel

### 🔄 Restablecer
- Un click para volver a valores originales
- No afecta las anotaciones

## 📏 Herramientas de Visualización

### ▦ Cuadrícula (Grid)
- Retícula de 50px
- Ayuda con simetría
- Referencias de proporción
- Toggle on/off con indicador visual

### 📐 Reglas
- Reglas en bordes superior e izquierdo
- Marcas cada 50px
- Coordenadas precisas
- Profesional como software CAD

## 🎯 Casos de Uso Específicos

### Rinoplastia:
1. Usar **medición** para altura nasal
2. **Líneas** para perfil deseado
3. **Flechas** para dirección de cambios
4. **Números** para marcar zonas (1: dorso, 2: punta, 3: columela)

### Lifting Facial:
1. **Cuadrícula** activada para simetría
2. **Flechas** para vectores de tracción
3. **Números** para puntos de sutura
4. **Dibujo libre** para marcar incisiones

### Botox/Toxina Botulínica:
1. **Números** para puntos de inyección
2. **Texto** con unidades por punto
3. **Círculos** para áreas de difusión
4. **Medición** para distancias entre puntos

### Rellenos Faciales:
1. **Rectángulos** para zonas a rellenar
2. **Flechas** indicando dirección de aplicación
3. **Números** para secuencia de inyección
4. **Texto** con tipo y ml de producto

### Blefaroplastia:
1. **Ajustar contraste** para ver pliegues
2. **Dibujo libre** para marcar exceso de piel
3. **Líneas** para incisiones planificadas
4. **Medición** para cantidad de piel a resecar

## 🚀 Flujo de Trabajo Profesional

### 1. Preparación
- Capturar o subir imagen en buena iluminación
- Ajustar brillo/contraste si es necesario
- Activar cuadrícula para simetría

### 2. Análisis
- Usar mediciones para proporciones
- Activar reglas para referencias
- Marcar puntos clave con números

### 3. Planificación
- Dibujar líneas de incisión con freehand
- Marcar zonas con círculos/rectángulos
- Indicar direcciones con flechas

### 4. Documentación
- Agregar textos descriptivos
- Verificar todas las marcas
- Descargar y guardar

### 5. Presentación al Paciente
- Imagen clara con anotaciones
- Explicar cada número/marca
- Documentar consentimiento

## 💡 Tips Profesionales

1. **Siempre usar cuadrícula** para procedimientos que requieren simetría
2. **Numerar en sentido horario** para facilitar seguimiento
3. **Colores consistentes** por tipo de procedimiento
4. **Guardar múltiples versiones** para comparar opciones
5. **Texto breve y claro** (unidades, medidas, nombres)
6. **Mediciones en puntos clave** antes de cualquier marca
7. **Dibujo suave** para aspecto más profesional
8. **Ajustar imagen primero**, anotar después

## 🎓 Casos de Estudio

### Ejemplo 1: Rinomodelación con Ácido Hialurónico
1. Activar cuadrícula y reglas
2. Medición: altura nasal actual
3. Números 1-6: puntos de inyección
4. Texto en cada número: cantidad en ml
5. Flecha: dirección de modelado
6. Antes/después simulado con ajustes de contraste

### Ejemplo 2: Protocolo de Toxina Facial Completo
1. Zoom 150% para mayor precisión
2. Números 1-30: todos los puntos de inyección
3. Colores: Rojo (frente), Azul (periocular), Verde (tercio inferior)
4. Texto con unidades por área
5. Círculos para zonas de difusión
